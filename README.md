# Python Send Email Standard Library

This example demonstrates how to send an email using only Python's built-in `smtplib` and `email.message` modules. It showcases sending a plain-text email to an SMTP server without relying on any third-party libraries. Users must provide sender and recipient credentials via environment variables.

## Language

`python`

## How to Run

1. Set `SENDER_EMAIL`, `SENDER_PASSWORD` (use an app password for Gmail), and `RECIPIENT_EMAIL` environment variables.
2. Run the script: `python main.py`
